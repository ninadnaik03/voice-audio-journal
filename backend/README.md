# Sunset Journal Voice Auth

Private FastAPI speaker-verification service for Sunset Journal.

Configure `MONGODB_URI`, `MONGODB_DATABASE`, `FRONTEND_ORIGIN`,
`SESSION_COOKIE_SECURE=true`, and `SESSION_COOKIE_SAMESITE=none` as Space
secrets or variables.
